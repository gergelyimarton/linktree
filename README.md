# Linktree

A personal portfolio and link aggregation website for Marton Gergelyi — a Hungarian content creator, musician, and developer. The site serves as a central hub to access music projects, interactive tools, and more.

## Tech Stack

- **HTML5** with semantic markup
- **CSS3** — glassmorphism effects, custom properties, responsive design
- **Vanilla JavaScript (ES6+)** — no framework dependencies
- **Canvas API** — particle effects
- **Font Awesome 6.0** — icons
- **Google Fonts (Montserrat)** — typography

## Features

### Visual Effects
- Dynamic gradient background with mouse-following animation (desktop)
- Interactive particle system with mouse repulsion
- Glare hover effect on cards
- Loading screen with resource tracking

### Pages
- **Main** — profile card with navigation to music projects
- **Nitebork** — music producer/DJ profile with SoundCloud embed
- **Duplajézus** — rapper profile with SoundCloud & YouTube embeds
- **Projects** — interactive tools portfolio

### Interactive Tools
- Personality Test (Moon vs Sun)
- AI Workout Generator
- Love Trial (date rating)
- Sleep Cycle Calculator

### Responsive Design
- Desktop-first with three breakpoints (940px, 768px, 550px)
- Heavy effects disabled on mobile for performance
- Touch-friendly mobile interface
